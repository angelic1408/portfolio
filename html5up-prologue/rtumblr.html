Designing a Developer Blog

Role: Designer, Developer

The site can be accessed here


This tumblr blog is part of our feature which was released in early 2016. The purpose for this tumblr blog is to create a media where developers could get information about the newest features or any important information such as maintenance and so on.

Problem

Tumblr theme is slightly different from the normal html. Tumblr is built around seven key post types: text, photos, links, quotes, chat, audio, and video, and each of them has slighly different parameters and customization.
One of the requirement was to make the appearance not look similar to the tumblr basic style. It requires me to hide several features of tumblr such as like and reblog.
Last problem: the internal company analytics tool can't be used for 3rd party content provider.


Solution


Create a script to hide tumblr iFrame. I added another script to retrive all the used tags and retrieve the latest posts.
For the analytics tool, I added Google Analytics code as replacement for the internal tool.
I worked extra miles to create the tumblr blog responsive.


Learned Lesson


The blog is live and it's constantly updated. The page speed is good and so the UX score. The site receive a positive feedback from the user when it was launched. Until now, it still gets hundreds of page view and nev visitors every month.
If I did not have enough time to tweak all post types from tumblr.If possible, I would like to style the other post such as photo and photosets post.
