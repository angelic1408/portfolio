Overview

The site can be accessed here


This tumblr blog is part of our feature which was released in early 2016. The purpose for this tumblr blog is to create a media where developers could get information about the newest features or any important information such as maintenance and so on.

There was some discussion about what kind of CMS should be use to achieve that goal. Some may say wordpress or blogspot. However, due to the limited time of the project, tumblr blog is chosen for its simplicity and ease of use.

Based on the analytics tool, this site is doing well. I managed to achieve stable sessions (user sessions) every months since it was released.


The Challenge


One of the challenge on this project is tumblr theme is slightly different from the normal html. Tumblr is built around seven key post types: text, photos, links, quotes, chat, audio, and video, and each of them has slighly different parameters and customization, for example, picture post can't have a title and chat post need to be structured in data list.

In addition to that, one of the requirement is to make the appearance does not look like tumblr basic style. It requires me to hide several features of tumblr such as like and reblog. I also need to create several script to hide tumblr iFrame.

I applied google anyaltics in this project because the internal analytics tool can't be used for 3rd party content provider.


Responsive


I worked extra miles to create the tumblr blog responsive. It was not required in the project, but I think most people check social media with their phone and it would be very nice to have a clean look even in smart phone.

Several javaScripts are use to achive that goal. By retrieving all the used tags, whenever user click a tag (in the sidebar or below the post title), the user will be redirected to all posts related with the tag. This will save users' time because they don't need to search for irrelevant words.
